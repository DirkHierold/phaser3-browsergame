# Left or Right

This is the game folder for **Left or Right**.

- Central game name variable: see `gameName.ts` for easy renaming.
- Entry point: `main.ts`
- HTML: `index.html`
- Manifest: `manifest.json`
- Styles: `game.css`
- Scenes: `scenes.ts`

To change the game name everywhere, edit the value in `gameName.ts`.
